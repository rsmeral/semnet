ArtNet Semantic network of artworks

SemNet can be downloaded either in a compiled version along with all required libraries and a sample network configuration (ArtNet), or only as source files and documentation.

semnet_artnet_conf.zip (12 MB)

SemNet binaries, required libraries, sources, documentation, ArtNet configuration files

semnet_src.zip (1 MB)

SemNet sources (+NetBeans projects and Ivy files), documentation

The dependencies can be managed automatically using Ivy which downloads all dependencies defined in the provided Ivy files at build time.
The libraries used are:

  • Sesame
  • HTMLCleaner
  • XStream
  • juniversalchardet
  • (PostgreSQL) JDBC driver for the HTMLCrawler's link database
